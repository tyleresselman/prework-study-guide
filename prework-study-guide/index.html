HTML

  • HTML is the skeleton, CSS is the skin?
  • The head element contains the metadata about the page. The body is what the user sees rendered.
  • Parent elements have child elements nested within them.

CSS

  • CSS gives a page its structure and style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Can use a class selector using a '.' before an item in the style sheet.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Loops are used to perform repeated tasks based on a condition.
  • A function allows you to define a block of code, name it, and execute it as many times as you want.
